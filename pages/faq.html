Dúvidas Comuns

Como faço para cadastrar uma nova tarefa?

Basta acessar a página "Cadastro de tarefa" no menu inicial, preencher a descrição, definir a prioridade e clicar em "Cadastrar tarefa".

O campo de data limite é obrigatório?

Não. O campo de data é totalmente opcional, ideal para atividades que não possuem um prazo rígido de entrega.

Como faço para deletar uma tarefa?

Para excluir uma tarefa, acesse a listagem na página inicial ou na tabela e clique no botão cinza "Excluir" ao lado dos dados da atividade.
