# Gerenciador de Tarefas

Sistema web para cadastro, listagem, edição e exclusão de tarefas. Projeto desenvolvido como parte de um curso de desenvolvimento web full stack, partindo do zero absoluto até a integração completa com banco de dados.

## 🚀 Funcionalidades

- Cadastro de tarefas com descrição, prioridade, status e data limite
- Listagem de tarefas em tabela com badges de prioridade
- Histórico de tarefas concluídas
- Busca por palavras-chave
- Formulário de contato
- FAQ interativo com seções expansíveis
- Validação de formulários com feedback visual em tempo real
- Layout responsivo (desktop e mobile)

## 🛠️ Tecnologias

| Camada | Tecnologia |
|--------|------------|
| Estrutura | HTML5 (semântico) |
| Estilo | CSS3 (flexbox, variáveis, design responsivo) |
| Lógica no navegador | JavaScript (validação, DOM, eventos) |
| Back-end | PHP (em breve) |
| Banco de dados | MySQL (em breve) |
